# مشروع Pac-Man - خوارزميات الذكاء الاصطناعي

مشروع لعبة Pac-Man يستخدم 8 خوارزميات مختلفة للبحث عن المسار، مع نظام اختبار لقياس الأداء وإنشاء واجهة رسومية للخوارزمية الأفضل.

## هيكل المشروع

```
project/
├── algorithms/          # مجلد الخوارزميات (8 خوارزميات)
│   ├── __init__.py
│   ├── base.py          # الدوال الأساسية المشتركة
│   ├── astar.py         # خوارزمية A*
│   ├── dijkstra.py      # خوارزمية Dijkstra
│   ├── bfs.py           # خوارزمية BFS
│   ├── dfs.py           # خوارزمية DFS
│   ├── greedy.py        # خوارزمية Greedy Best-First
│   ├── bidirectional.py # خوارزمية Bidirectional Search
│   ├── ida_star.py      # خوارزمية IDA*
│   └── theta_star.py    # خوارزمية Theta*
├── tests/               # نظام الاختبار
│   ├── __init__.py
│   ├── algorithm_tester.py  # فئة اختبار الخوارزميات
│   └── run_tests.py     # سكريبت تشغيل الاختبارات
├── gui/                 # واجهة المستخدم الرسومية
│   ├── __init__.py
│   └── game_gui.py      # واجهة اللعبة
├── maps/                # الخرائط
│   └── map1.txt
├── game.py              # اللعبة الأساسية (للتوافق)
├── ghost.py             # كلاس الأشباح
├── player.py            # كلاس اللاعب
├── map.py               # كلاس الخريطة
├── main.py              # نقطة الدخول الرئيسية
├── run_tests.py         # تشغيل الاختبارات
└── requirements.txt    # المتطلبات

```

## متطلبات التشغيل

1. تثبيت Python 3.9 أو أحدث
2. تثبيت المكتبات المطلوبة:
```bash
pip install -r requirements.txt
```

## الخوارزميات المطبقة

المشروع يحتوي على 8 خوارزميات للبحث عن المسار:

1. **A*** - خوارزمية البحث الأكثر استخداماً
2. **Dijkstra** - خوارزمية البحث عن المسار الأقصر
3. **BFS** - البحث بالعرض أولاً
4. **DFS** - البحث بالعمق أولاً
5. **Greedy Best-First** - البحث الجشع
6. **Bidirectional Search** - البحث ثنائي الاتجاه
7. **IDA*** - Iterative Deepening A*
8. **Theta*** - Any-Angle Pathfinding

## كيفية الاستخدام

### 1. تشغيل الاختبارات

لتشغيل الاختبارات وإيجاد أفضل خوارزمية:


python run_tests.py


سيقوم هذا الأمر بـ:
- اختبار جميع الخوارزميات الـ8
- قياس معدل النجاح، طول المسار، والسرعة
- تحديد أفضل خوارزمية بناءً على الأداء
- حفظ النتائج في ملف `best_algorithm.json`

### 2. تشغيل اللعبة

بعد تشغيل الاختبارات، يمكنك تشغيل اللعبة باستخدام أفضل خوارزمية:

1/ python main.py


## نظام الاختبار

نظام الاختبار يقيس:
- **معدل النجاح**: نسبة نجاح الخوارزمية في إيجاد مسار
- **متوسط طول المسار**: متوسط عدد الخطوات في المسار
- **متوسط الوقت**: متوسط الوقت المستغرق للبحث

يتم حساب النقاط بناءً على:
- معدل النجاح (50%)
- كفاءة المسار (30%)
- السرعة (20%)
