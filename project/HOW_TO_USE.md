# دليل الاستخدام - مشروع Pac-Man

## الخطوات المطلوبة

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل الاختبارات (مهم جداً!)

**يجب تشغيل الاختبارات أولاً** لتحديد أفضل خوارزمية:

```bash
python run_tests.py
```

هذا الأمر سيقوم بـ:
- اختبار جميع الخوارزميات الـ8
- قياس الأداء والدقة لكل خوارزمية
- تحديد أفضل خوارزمية بناءً على النتائج
- حفظ النتائج في ملف `best_algorithm.json`

**ملاحظة مهمة**: الواجهة الرسومية تستخدم تلقائياً أفضل خوارزمية من نتائج الاختبارات.

### 3. تشغيل اللعبة

بعد تشغيل الاختبارات، يمكنك تشغيل اللعبة:

```bash
python main.py
```

أو:

```bash
python gui/game_gui.py
```

### 4. التحكم في اللعبة

- **↑** (السهم للأعلى) - التحرك للأعلى
- **↓** (السهم للأسفل) - التحرك للأسفل  
- **←** (السهم لليسار) - التحرك لليسار
- **→** (السهم لليمين) - التحرك لليمين

## الخوارزميات المطبقة

1. **A*** - خوارزمية البحث الأكثر استخداماً
2. **Dijkstra** - خوارزمية البحث عن المسار الأقصر
3. **BFS** - البحث بالعرض أولاً
4. **DFS** - البحث بالعمق أولاً
5. **Greedy Best-First** - البحث الجشع
6. **Bidirectional Search** - البحث ثنائي الاتجاه
7. **IDA*** - Iterative Deepening A*
8. **Theta*** - Any-Angle Pathfinding

## هيكل المشروع

- `algorithms/` - مجلد يحتوي على جميع الخوارزميات الـ8
- `tests/` - نظام الاختبار وقياس الأداء
- `gui/` - واجهة المستخدم الرسومية
- `maps/` - ملفات الخرائط
- `main.py` - نقطة الدخول الرئيسية
- `run_tests.py` - تشغيل الاختبارات
